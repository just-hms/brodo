# brodo

`br`anch t`odo` is a way to find out what needs to be done in your branch before is ready to be merged

usage:

```bash
cd /path/to/your/project
git checkout your-branch
brodo
```

## Install

```bash
go install github.com/just-hms/brodo@latest
```
